# Trading Application in Ruby on Rails

## Tech Stack
- Ruby on Rails
- PostgreSQL
- Docker
- Docker Compose
- RSpec

## Setup Instructions
1. Install Ruby 3.4.5
```
 rbenv install 3.4.5
```
2. Set Ruby 3.4.5 as local version
```
 rbenv local 3.4.5
```
3. Install Bundler
```
 gem install bundler
```
4. Install dependencies
```
bundle install
```
5. Run DB migrations
```
bin/rails db:migrate
```
6. Run tests
```
bin/rails test
```
